ld theme toggle

Body-attributet data-theme sätts till light eller dark.

Användning

Lägg till scriptet från CDN och placera komponenten där knapparna ska visas.

<script src="https://ld.j4rl.se/ld-theme-toggle.js" defer></script>

<ld-theme-toggle></ld-theme-toggle>

Styla sidan med attributet som komponenten sätter på body.

body[data-theme="light"] {
  background: #f8fafc;
  color: #111827;
}

body[data-theme="dark"] {
  background: #111827;
  color: #f8fafc;
}

Valt läge sparas automatiskt i localStorage. Som standard används systemets tema.
